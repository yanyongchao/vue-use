# useToggle

用于在两个状态值间切换的 Hook。

## API

```typescript
const [state, toggle] = useToggle(
  defaultValue?: boolean,
);
```

### Params

| 参数  | 说明  | 类型  | 默认值  |
| ------------ | ------------------------ | ------------------------------------------------ | ------- |
| defaultValue | 可选项，传入默认的状态值 | `boolean` | -

### Result

| 参数    | 说明     | 类型      |
| ------- | -------- | --------- |
| state   | 状态值   | -         |
| toggle | 切换状态函数 | `Function` |

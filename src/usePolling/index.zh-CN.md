## API

```typescript
let polling = usePolling()
let cancelPolling: Function = polling.startPolling({
  cb: Function, // 轮询函数
  time: number, // 轮询时间
  isAsync: boolean // 是否是同步
})
```

### Result

| 参数 | 说明                      | 类型                      |
|------|---------------------------|---------------------------|
| startPolling   | 轮询开始函数 | `(time = 1000, cb, isAsync = true) => Function` |

### cancelPolling
| 参数 | 说明                      | 类型                      |
|------|---------------------------|---------------------------|
| cancelPolling   | 轮询结束函数 | `Function` |
